# 🛒 E-Commerce Sales Dashboard (Power BI)

## 📌 Project Overview

This project is an end-to-end **E-Commerce Sales Analysis Dashboard** built using **Power BI**.
It focuses on analyzing sales performance, customer behavior, product trends, and profitability.

The dashboard provides **interactive visual insights** to help understand business growth and decision-making.

---

## 🎯 Key Objectives

* Analyze overall sales and profit performance
* Identify top-performing states and customers
* Understand category-wise and subcategory-wise trends
* Evaluate payment mode preferences
* Track monthly profit trends

---

## 📊 KPIs (Key Performance Indicators)

* 💰 **Total Amount (Sales)**
* 📦 **Total Quantity**
* 📈 **Total Profit**
* 🧮 **Average Order Value (AOV)**
